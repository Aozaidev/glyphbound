# Archived: Body Glyph Families

Superseded by [第一波-核心.md](../第一波-核心.md).

The old family-upgrade model (`心 -> 沁 -> 蕊`, `忍 -> 耐 -> 韧`, `净 -> 洁 -> 清`, `力 -> 气 -> 忿 -> 怒`) is no longer the first-wave design.

The current first-wave rule is:

```text
glyph = intent
staff tier = scale, duration, stability, and extra mechanics
```

Keep this file only as a compatibility pointer so older notes do not silently reintroduce the discarded upgrade-chain design.
